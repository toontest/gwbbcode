GWBBCode Javascript
===================

Originaly created in september 2011.

Example
============
```
<script src="https://toontest.github.io/gwbbcode/public/gwbbcode/api.js"></script>
<script type="text/javascript" >
    GWOnLoad(function(){
      GWBBCodeParse('.builds');
    });
</script>
<div class=".builds">
  [build=ADIRjh8oRyI5EQCd1e8RkJA]
</div>
```
